# Диспетчер задач (Tkinter + Pygame) 

Это учебный проект на Python (Вариант 11), демонстрирующий совместную работу двух графических библиотек: `Tkinter` (для панели управления) и `Pygame` (для визуализации процессов).

## Описание проекта
Программа симулирует распределение задач между двумя ядрами процессора. 
* В окне управления (Tkinter) пользователь решает, на какое ядро отправить текущую задачу.
* В окне визуализации (Pygame) отображается плавная анимация полета блока с задачей (с уникальным ID) к выбранному ядру.
* Реализован счетчик выполненных задач для каждого из ядер.

## Используемые технологии
* **Python 3**
* **Tkinter** (стандартная библиотека) — создание кнопок и вкладок.
* **Pygame** — отрисовка графики, анимация на базе векторов и игровой цикл.
* **ООП** — логика задач инкапсулирована в класс `TaskBlock`.
